CHAMPS DE DONNEES IDENTIFIES POUR CHAQUE PROPRIETE:
================================================================================

1. PRIX
   Description: Prix de la propriete
   Format: Montant avec ou sans taxes (TPS/TVQ)
   Exemples: "750 000 $", "1 050 000 $", "995 000 $ + TPS/TVQ", "1 389 000 $"
   Badges possibles: "Nouvelle annonce", "Nouveau prix"

2. ADRESSE
   Description: Adresse complete de la propriete
   Format: Numero et nom de rue, parfois avec plage (ex: 220Z-226BZ)
   Exemples: "220Z-226BZ Boul. Pierre-Bertrand", "164-174 Av. Ruel", "19 Rue Ste-Angèle"
   Note: Souvent affichee en gras

3. VILLE ET ARRONDISSEMENT
   Description: Localisation geographique
   Format: "Ville (Arrondissement)"
   Exemples: "Quebec (Les Rivieres)", "Quebec (Beauport)", "Quebec (La Cite-Limoilou)"

4. QUARTIER
   Description: Quartier specifique dans l'arrondissement
   Format: Nom du quartier
   Exemples: "Neufchatel-Est/Lebourgneuf", "Chutes-Montmorency", "Vieux-Quebec/Cap-Blanc/Colline parlem."
   Pattern dans le texte: "dans le quartier [NOM] construit en"

5. TYPE DE PROPRIETE
   Description: Type de batiment
   Format: Categorie de propriete
   Exemples: "Autre", "Quintuplex", "Duplex", "Triplex"
   Pattern dans le texte: "[TYPE] dans le quartier"

6. ANNEE DE CONSTRUCTION
   Description: Annee de construction du batiment
   Format: Annee (4 chiffres)
   Exemples: "1949", "1926", "1940", "1834"
   Pattern dans le texte: "construit en [ANNEE]"

7. NUMERO CENTRIS
   Description: Identifiant unique Centris
   Format: Numero a 8 chiffres
   Exemples: "23326443", "16311005", "24772371"
   Pattern dans le texte: "No Centris : [NUMERO]"

8. DATE D'ENVOI
   Description: Date de publication de l'annonce
   Format: YYYY-MM-DD
   Exemples: "2025-12-15", "2025-12-10", "2025-12-05"
   Pattern dans le texte: "Date d'envoi : [DATE]"

9. STATUT
   Description: Statut de l'annonce
   Format: Badge visuel ou absence de badge
   Valeurs possibles: "Nouvelle annonce", "Nouveau prix", ou aucun badge

10. ACTIONS UTILISATEUR
    Description: Actions disponibles pour chaque propriete
    Format: Boutons d'action
    Actions disponibles:
    - Enregistrer comme favori
    - Enregistrer comme possibilite
    - Enregistrer comme rejete
    - Retirer des favoris
    - Retirer des possibilites
    - Retirer des rejetes

11. LIEN VERS DETAIL
    Description: URL vers la page de detail de la propriete
    Format: Lien cliquable (probablement dans l'adresse ou le titre)
    Note: Non visible dans l'exemple, mais probablement disponible dans le HTML
